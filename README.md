# Water Quality Prediction Project

## Overview
End-to-end data project for water potability prediction using Python, SQL Server, Power BI and Streamlit.

## Technologies
- Python
- Pandas
- Scikit-learn
- SQL Server
- Power BI
- Streamlit

## Features
- Data cleaning and validation
- Random Forest prediction
- SQL analytical views
- Power BI dashboard
- Streamlit prediction app with domain warnings
